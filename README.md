Usage
The animation runs automatically when you open the page. The canvas element renders the heart animation with dynamic effects that create an engaging visual experience.

Customization
You can customize the animation by modifying:

Colors: Edit the color values in style.css or app.js

Animation Speed: Adjust timing parameters in app.js

Canvas Size: Modify width and height attributes in index.html
